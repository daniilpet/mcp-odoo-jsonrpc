# Architecture Decision Records

Данный каталог содержит записи архитектурных решений (ADR) проекта
MCP Odoo JSON-RPC в формате MADR (Markdown Any Decision Records).

## Реестр решений

| ADR   | Статус   | Решение                          |
|-------|----------|----------------------------------|
| ADR-001 | Accepted | Выбор языка реализации (Python)  |
| ADR-002 | Accepted | Стратегия аутентификации         |
| ADR-003 | Accepted | Паттерн интеграции с Odoo        |
| ADR-004 | Accepted | Dev-mode — разведка эндпоинтов   |
| ADR-005 | Accepted | Модель безопасности              |
| ADR-006 | Accepted | CLI → Service Layer (гибрид)     |

## Формат

Каждый ADR следует шаблону MADR:

- **Статус**: Proposed / Accepted / Deprecated / Superseded
- **Контекст**: описание проблемы
- **Рассмотренные варианты**: альтернативы с плюсами и минусами
- **Решение**: выбранный вариант с обоснованием
- **Последствия**: положительные, отрицательные, риски
